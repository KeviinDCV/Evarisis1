
# Informe del Proyecto (Foco: Gerente_irne_torres)

## Contexto del Proyecto
El proyecto es un sistema de registro de cáncer para el HUV (EVARISIS). Incluye un dashboard, múltiples agentes IA, y un sistema de extractores de datos (auditor_sistema.py) de PDFs de patología (IHQ). Un módulo clave analiza tendencias de biomarcadores para predecir la demanda futura de medicamentos de alto costo.

## Enfoque Principal: Registro cancer precision automatizacion servinte

El proyecto ofrece beneficios significativos alineados con los objetivos de la organización.
