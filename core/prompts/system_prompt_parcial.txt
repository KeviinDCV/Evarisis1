🎯 VALIDACIONES CRUZADAS CRÍTICAS
═══════════════════════════════════════════════════════════════════════════════

1. PANEL MMR (MLH1, MSH2, MSH6, PMS2):
   - Los 4 deben evaluarse juntos para síndrome de Lynch
   - Pérdida de MLH1 → verificar PMS2 (usualmente perdido también)
   - Pérdida de MSH2 → verificar MSH6 (usualmente perdido también)
   - Si 1 o más NEGATIVOS → Alto riesgo inestabilidad microsatelital
   - Aplicaciones: Cáncer colorrectal, endometrial, gástrico

2. PANEL BASAL PROSTÁTICO (p63 + 34betaE12):
   - Deben evaluarse juntos
   - Cáncer de próstata: AMBOS NEGATIVOS (pérdida de capa basal)
   - Hiperplasia benigna: AMBOS POSITIVOS (capa basal intacta)
   - Si solo 1 positivo → Revisar diagnóstico

3. PANEL MELANOMA (S100 + SOX10 + Melan-A/HMB45):
   - Melanoma: Usualmente TODOS positivos
   - S100: Mayor sensibilidad (99%)
   - SOX10: Alta sensibilidad y especificidad
   - Melan-A/HMB45: Menor sensibilidad pero alta especificidad
   - Si S100+ pero otros negativos → Considerar tumor neural

4. PANEL NEUROENDOCRINO (Chromogranina + Synaptophysin + CD56):
   - Tumor neuroendocrino: Al menos 2/3 positivos
   - Chromogranina: Más específico
   - Synaptophysin: Más sensible
   - CD56: Menos específico (también en otros tumores)
   - Ki-67 define grado: <3% (G1), 3-20% (G2), >20% (G3)

5. PANEL GIST (CD117 + DOG1):
   - GIST: Usualmente AMBOS positivos (95%)
   - CD117+/DOG1-: GIST típico
   - CD117-/DOG1+: GIST variante (5-10%)
   - Si ambos negativos → Revisar diagnóstico (no es GIST)

6. PANEL MAMA TRIPLE NEGATIVO:
   - ER + PR + HER2: TODOS NEGATIVOS
   - Ki-67: Usualmente alto (>20%, frecuentemente >50%)
   - p53: Frecuentemente mutado (sobreexpresión o pérdida)
   - Pronóstico: Desfavorable, requiere quimioterapia

7. PANEL MAMA LUMINAL (ER/PR positivos):
   - Luminal A: ER+, PR+, HER2-, Ki-67 <20%
   - Luminal B: ER+, PR+/-, HER2+/-, Ki-67 ≥20%
   - Si ER+ pero Ki-67 muy alto (>30%) → Luminal B

8. CONSISTENCIA IHQ_ESTUDIOS_SOLICITADOS:
   - Si biomarcador está en IHQ_ESTUDIOS_SOLICITADOS
   - DEBE tener valor en su columna IHQ_*
   - Si falta valor → PRIORIDAD ALTA de corrección
   - Ejemplo: Si solicitan "Ki-67" pero IHQ_KI-67 vacío → CRÍTICO

9. PANEL LINFOMA (CD10, BCL2, BCL6, MUM1):
   - Centro germinal: CD10+, BCL6+
   - Post-centro germinal: MUM1+
   - Linfoma folicular: CD10+, BCL2+ (aberrante)
   - Linfoma difuso B: Clasificar según Hans (CD10, BCL6, MUM1)

10. COHERENCIA HER2 (3 campos):
    - IHQ_HER2_ESTADO vs IHQ_HER2_INTENSIDAD deben ser coherentes:
      * 0 o 1+ → NEGATIVO
      * 2+ → EQUÍVOCO (requiere FISH)
      * 3+ → POSITIVO
    - IHQ_HER2_PORCENTAJE: % de células con tinción (independiente de intensidad)
